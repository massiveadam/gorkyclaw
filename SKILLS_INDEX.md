# Skills Index

This repo stores shareable runtime skills in `skills/`.

## Included

- `skills/templates/basic-skill`: starter template for new skills.

## Addon Packs

- `addons/image-to-text`: OCR capability scaffold
- `addons/voice-to-text`: speech-to-text scaffold
- `addons/opencode-serve`: OpenCode `/serve` integration scaffold

## Add a new skill

1. Copy `skills/templates/basic-skill` to `skills/<your-skill-name>`.
2. Edit `SKILL.md`.
3. Add your new entry to this file.
